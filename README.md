# Domain Recon Tool

A Python-based OSINT reconnaissance tool for extracting WHOIS records, DNS information, and hosting IP intelligence from target domains.

## Features

- WHOIS Lookup
- DNS Record Enumeration
- IP Address Resolution
- Rich Terminal Output

## Installation

```bash
pip install python-whois dnspython rich
```

## Usage

```bash
python DomainReconTool.py
```

## Example

Enter Target Domain Name:

google.com
